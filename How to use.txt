==========================================
TabOS Pro 使用說明（最終版）
==========================================

👉 核心目標：
「開機 → 一個畫面 → 管全部分頁 + 工作知識」

==================================================
一、安裝外掛（Opera / Chrome / Edge）
==================================================

1. 開啟瀏覽器

Opera：
opera://extensions

Chrome：
chrome://extensions

Edge：
edge://extensions

2. 開啟「開發人員模式」

3. 點擊：
👉 載入未封裝項目（Load unpacked）

4. 選擇資料夾：
👉 opera-tabs-manager/extension

❗ 注意：
不要選整個專案，只選 extension

==================================================
二、備份目前所有分頁（最常用）
==================================================

👉 點外掛 → 按：

👉 「備份目前所有分頁」

輸出：

tabos_pro_tabs_YYYY-MM-DD.html

👉 用瀏覽器打開即可開始管理

==================================================
三、管理 HTML（最重要）
==================================================

打開 HTML 後你可以：

✔ 開啟分頁
✔ 標記 已看 / 未看
✔ 設定狀態：
   - 待看
   - 重要
   - 工作
   - 深度閱讀

✔ 設定優先度：
   - P1（最重要）
   - P2
   - P3

✔ 加 tags（AI, 投資, 技術...）

✔ 搜尋
✔ 篩選分類
✔ 只看未看
✔ 匯出整理後 HTML

👉 群組功能（超重要）：

✔ 本群全標已看
✔ 本群全標未看
✔ 開本群未看
✔ 刪本群已看

==================================================
四、舊 HTML 轉新版（超重要）
==================================================

支援：

tabs_backup.html
tabs_backup_managed_xxx.html
任何有 <a href> 的頁面

👉 操作：

1. 點外掛
2. 選擇舊 HTML
3. 點：

👉 「轉成新版可管理 HTML」

✔ 自動分類
✔ 自動去重
✔ 變成可操作 UI

==================================================
五、🔥 合併舊 HTML + 新分頁（核心功能）
==================================================

👉 使用情境：

昨天已存 HTML
今天又開了一堆新分頁

👉 解法：

1. 點外掛
2. 選擇舊 HTML
3. 點：

👉 「合併目前分頁並匯出新版 HTML」

系統會：

✔ 讀舊 HTML
✔ 抓目前所有分頁
✔ 合併
✔ 去重
✔ 保留已看狀態
✔ 自動分類

👉 可選：

☑ 合併後關閉目前分頁（推薦）

==================================================
七、🔥 Worklog Manager（工程神器）
==================================================

👉 功能：
把「工作日誌.txt」變成可搜尋知識庫

------------------------------------------

📌 使用方式：

1. 把工作日誌放進：

opera-tabs-manager/

檔名：

工作日誌.txt
或
command.txt

2. 執行：

👉 run_worklog.bat

------------------------------------------

📌 或手動：

python tools\worklog_parser.py 工作日誌.txt output\worklog.html

------------------------------------------

📌 輸出：

output/worklog.html

用瀏覽器打開即可

------------------------------------------

📌 功能：

✔ 日期 timeline
✔ 搜尋
✔ 分類
✔ 指令高亮
✔ TODO 抽取
✔ IP 高亮
✔ 統計
✔ 可編輯

------------------------------------------

📌 可編輯內容：

✔ 修改文字
✔ 修改日期
✔ 修改分類
✔ 切換 TODO
✔ 切換 command

✔ 匯出修改後 HTML

==================================================
八、UI 操作技巧（你會用很爽）
==================================================

👉 快速整理流程：

1. 開 HTML
2. 搜尋
3. 標記已看
4. 刪除垃圾
5. 匯出乾淨版

👉 每天只看：

☑ 只看未看

👉 分類用：

AI / 工作 / 投資

👉 P1 用來：

👉 當天一定要看

==================================================
九、常見問題（很重要）
==================================================

Q：為什麼沒有反應？
👉 extension 沒 reload

Q：icon 錯誤？
👉 icons 裡面要有 png

Q：HTML 打不開？
👉 用瀏覽器開，不是記事本

Q：run_worklog 失敗？
👉 路徑錯誤（tools / parser）

==================================================
十、建議使用流程（最強）
==================================================

每天：

1. 開一堆分頁
2. 備份
3. 合併舊 HTML
4. 清理
5. 匯出

👉 永遠只有一份乾淨資料
