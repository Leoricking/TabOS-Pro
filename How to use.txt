==========================================
Tabs Manager Pro 使用說明
==========================================

👉 核心目標：
「開機 → 一個畫面 → 管全部分頁 + 每日任務 + 工作知識 + 股票策略 + 資料同步」

Tabs Manager Pro 不是單純分頁工具。

它是：

👉 個人知識作業系統
👉 工程師工作整理中心
👉 分頁知識庫
👉 Daily Planner
👉 Worklog Manager
👉 Trading Mode
👉 Sync Center

==========================================
一、安裝外掛（Opera / Chrome / Edge）
==========================================

1. 開啟瀏覽器擴充功能頁面

Opera：

opera://extensions

Chrome：

chrome://extensions

Edge：

edge://extensions

2. 開啟「開發人員模式」

3. 點擊：

👉 載入未封裝項目
👉 Load unpacked

4. 選擇資料夾：

👉 opera-tabs-manager/extension

❗ 注意：

請選 extension 資料夾

不要選整個專案根目錄

正確：

opera-tabs-manager/extension

錯誤：

opera-tabs-manager

==========================================
二、外掛主畫面功能
==========================================

點擊瀏覽器工具列上的 Tabs Manager Pro 圖示後，會看到：

1. 備份目前所有分頁

2. 合併舊 HTML + 目前分頁

3. 舊版 HTML 轉新版

4. Daily Planner

5. Trading Mode

6. Sync Center

7. Pro 功能說明

==========================================
三、備份目前所有分頁（最常用）
==========================================

👉 點外掛 → 按：

👉 備份目前所有分頁

輸出檔案格式：

tabs_manager_pro_tabs_YYYY-MM-DD_HH-MM-SS.html

用瀏覽器打開即可開始管理。

你可以選：

☑ 備份後關閉目前一般網頁分頁

用途：

當你開太多分頁時，可以先全部備份成 HTML，再關閉目前分頁，讓瀏覽器變乾淨。

十、Trading Mode（台股自選股 / 操作策略）
==========================================

Trading Mode 是台股策略與盤中決策記錄面板。

它不是自動下單工具。

它的用途是：

✔ 整理自選股
✔ 紀錄當沖策略
✔ 紀錄短線策略
✔ 紀錄中線策略
✔ 紀錄長線策略
✔ 紀錄支撐 / 壓力 / 停損 / 停利
✔ 接 Python Engine 回測訊號
✔ 將交易提醒轉成 Planner 任務

------------------------------------------
開啟方式
------------------------------------------

點外掛：

👉 開啟台股自選股 / 操作策略

------------------------------------------
股票資料
------------------------------------------

每檔股票可以填：

✔ 股票代號
✔ 股票名稱
✔ 市場
✔ 現價
✔ 漲跌幅
✔ 優先級
✔ 主評級
✔ tags

------------------------------------------
策略區塊
------------------------------------------

支援四種週期：

1. 當沖

適合今日盤中，不留倉。

2. 短線

約 1～10 個交易日。

3. 中線

約 2～8 週。

4. 長線

約 3～12 個月以上。

------------------------------------------
回測信心指標
------------------------------------------

可記錄：

EV
Score
Win Rate
Avg Profit
Avg Loss
Profit Factor
Sharpe
Max Drawdown
Similarity
Confidence

------------------------------------------
EV 公式
------------------------------------------

EV = (Win Rate × Avg Profit) - (Loss Rate × Avg Loss)

用途：

判斷策略期望值是否為正。

------------------------------------------
觸發顏色
------------------------------------------

進入買區：

黃色

失效 / 停損：

紅色

EV > 0 且 Score >= 70：

綠色

------------------------------------------
Trading Priority
------------------------------------------

P1：

今日盤中一定要盯

例如：

- 進入買區
- 接近停損
- 突破壓力
- 高信心訊號

P2：

本週追蹤，等待條件成立。

P3：

長期觀察，不急。

------------------------------------------
轉 Planner 任務

點：

👉 轉 Planner 任務

會自動建立今日任務，內容包含：

十一、Python Engine API Sync
==========================================

Trading Mode 可以對接本機 Python 回測 / 報價引擎。

預設 endpoint：

http://127.0.0.1:8787

------------------------------------------
建議 API
------------------------------------------

GET /health

GET /signals

GET /signal/{symbol}

------------------------------------------
操作方式
------------------------------------------

1. 啟動 Python Engine

2. 確認 API 跑在：

http://127.0.0.1:8787

3. 打開 Trading Mode

4. 在 API Endpoint 輸入：

http://127.0.0.1:8787

5. 點：

👉 儲存 Endpoint

6. 點：

👉 檢查連線

7. 點：

👉 一鍵同步後端訊號

------------------------------------------
Python Engine 回傳範例
------------------------------------------

{
  "signals": [
    {
      "symbol": "2308",
      "name": "台達電",
      "price": 2165,
      "changePercent": -1.2,
      "priority": "P1",
      "status": "WATCH_BUY_ZONE",
      "action": "等待回測",
      "buyZone": "2150-2160",
      "stopLoss": "跌破 2150 拉不回",
      "takeProfit": "2220-2250",
      "risk": "中",
      "ev": 2.07,
      "score": 70,
      "winRate": 57,
      "profitFactor": 1.45,
      "maxDrawdown": 16,
      "sharpe": 1.25,
      "confidence": 68,
      "message": "現價接近短線第一買區，等待支撐確認。"
    }
  ]
}

------------------------------------------
注意
------------------------------------------

券商 API Key 不要放在 Extension 裡。

正確架構：

券商 API Key
→ Python 後端
→ FastAPI / Flask
→ Extension fetch localhost API

Extension 只負責顯示與提醒。

==========================================
十四、常見問題
==========================================

Q：外掛修改後沒有變化？

A：

請到 extensions 頁面重新載入外掛。

不要只按 F5。

------------------------------------------

Q：Planner 後一天 / 前一天沒反應？

A：

1. 重新載入 extension
2. 關掉舊 Planner 頁籤
3. 從 popup 重新開 Planner

------------------------------------------

Q：無法載入 extension？

A：

請確認選的是：

opera-tabs-manager/extension

不是：

opera-tabs-manager

------------------------------------------

Q：icon 錯誤？

A：

icons 裡面要有 manifest 指定的 png。

------------------------------------------

Q：HTML 打不開？

A：

請用瀏覽器開，不是用記事本。

------------------------------------------

Q：run_worklog 失敗？

A：

確認路徑是：

tools/worklog_parser.py

不是 parser/worklog_parser.py

------------------------------------------

Q：Sync Center 顯示無資料？

A：

代表目前 localStorage 還沒有 Planner / Trading / Tabs 資料。

先建立資料，再回 Sync Center 匯出。

------------------------------------------

Q：Trading API 連不上？

A：

檢查：

1. Python Engine 是否啟動
2. endpoint 是否為 http://127.0.0.1:8787
3. /health 是否可正常回應
4. 瀏覽器是否擋 CORS
